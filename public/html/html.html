    Bienvenue au cours HTML et CSS

    Ce cours vous apprendra à créer des sites web avec HTML et CSS, en suivant les bonnes pratiques.

    À propos du cours

    Ce cours est conçu pour les débutants en développement web. Vous apprendrez comment créer une structure de base avec HTML et comment styliser vos pages avec CSS.

    Objectifs du cours

    • Comprendre la structure d'un fichier HTML.
    • Utiliser les balises HTML courantes.
    • Apprendre les bases du CSS pour styliser vos pages.
    • Construire une page web simple en HTML et CSS.
